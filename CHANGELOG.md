#### [v1.1.1](https://github.com/HackNotice/n8n-nodes-hacknotice/compare/v1.1.0...v1.1.1)

> 13 May 2026

- **HackNotice MCP API**: MCP endpoint is fixed in code (no URL field in the UI);
  **HackNotice MCP** HTTP transport sends JSON-RPC as a serialized body and uses
  a 120s HTTP timeout to avoid dropped connections with n8n’s request helper.
- **HackNotice MCP** default endpoint URL is `https://mcp.hacknotice.com:13330/mcp`.

#### v1.1.0

- Added a new **HackNotice MCP** node that connects to a running
  `hacknotice-mcp-server` over Streamable HTTP and exposes every tool the
  server reports (`tools/list`) as a dynamic dropdown. Operations:
  `Call Tool` (forwards a JSON arguments object to `tools/call`) and
  `List Tools`.
- Added a new **HackNotice MCP API** credential (`hackNoticeMcpApi`) with
  fields for the MCP endpoint URL and the per-user integration key (sent as
  `X-HackNotice-Integration-Key`). Includes a credential test that performs a
  JSON-RPC `initialize` round-trip.
- The HackNotice **node** now exposes the full **Assessments** area of the extensions API (matching the Postman Assessments folder): assessments, assessment events (customer and invited), templates, preferences, invites, and assessment data files (customer and invited). Use **Document ID**, **Page Number**, **Invite Code**, and **Request Body** as appropriate for each operation; file downloads return binary data; invited uploads use multipart with an input binary field.
- Added **Third Party Watchlist** operations to the HackNotice node: **Get Watchlist Domains** (`POST /hackwatchlist/page/:pageNum`) and **Search Domain** (`POST /hackwatchlist/search`).
- Added Third Party Watchlist ID operations: **Get By ID** (`GET /hackwatchlist/:id`), **Update By ID** (`PUT /hackwatchlist/:id`), and **Delete By ID** (`DELETE /hackwatchlist/:id`).
- Added **First Party Watchlist** operations to the HackNotice node: **Get Watchlist Items** (`POST /domainwatchlist/page/:pageNum`), **Search Item** (`POST /domainwatchlist/search`), and **Create Item** (`POST /domainwatchlist/create`).
- Added First Party Watchlist ID operations: **Get By ID** (`GET /domainwatchlist/:id`) and **Delete By ID** (`DELETE /domainwatchlist/:id`).
- Added **End User Watchlist** operations to the HackNotice node: **Get Many** (`GET /enduserwatchlist/page/:pageNum`), **Search for Email** (`POST /enduserwatchlist/search`), **Add Item to Watchlist** (`POST /enduserwatchlist/create`), **Get By ID** (`GET /enduserwatchlist/:id`), and **Delete By ID** (`DELETE /enduserwatchlist/:id`).
- Updated the **HackNotice API** credential to support either API key/email/password authentication or an integration key sent as `X-HackNotice-Integration-Key`.
